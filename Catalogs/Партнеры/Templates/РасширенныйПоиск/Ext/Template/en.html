Advanced search is performed not only by the Partners catalog but also by its related data, such as the Contact persons, Contracts, and other catalogs.

This search provides a wide range of capabilities. At the same time, it uses a query language of the full-text search, so certain results might seem unexpected.

For example, to search by a part of a string, you need to add an asterisk (*). If you enter "docum", the word "document" will not be found. Instead, you need to enter "docum*" in the search string.

You can search by several words, use search operators, or enter an exact phrase.

You can use the following search operators in the search string:

Example 2:
sale NEAR/+3 item

Example 3:
sale NEAR/-3 item

 Search for data that contains specified words separated by n words in one attribute (considering the morphology).
The sign indicates in which direction from the first word the second word will be searched for ("+" means after the first word, "-" means before the first word).
If the sign is not specified, the search will find data that contains the specified words within n words both before and after the first word. The word order does not matter.
In example 1, the search will find data where the word "item" is located no further than 3 words before or after the word "sale".
In example 2, the search will find data where the word "item" is located no further than 3 words after the word "sale".
In example 3, the search will find data where the word "item" is located no further than 3 words before the word "sale".
